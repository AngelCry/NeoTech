What is Neotech?

NeoTech is an attempt to create a new kind of tech mod in the new 1.8.9 environment. From pipes, to generators, to machines we aim to bring all the best of older tech mods and give them a new light. Since we started in this version of Minecraft, we have the benefit of not having to port existing code. This allows us to create clean code specifically designed for this new version of Forge. Lots of thing have changed and we will always keep up with the latest changes. This includes new things like Item/Block models, IItemHanders, and whatever else comes our way.
